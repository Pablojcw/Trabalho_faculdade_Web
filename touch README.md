# Sistema de Lista de Tarefas

Projeto desenvolvido para a disciplina de Programação Web.

## Descrição

Este sistema permite cadastrar e gerenciar tarefas do dia a dia.
As tarefas podem ser criadas, pesquisadas, editadas e excluídas.
Os dados são armazenados utilizando LocalStorage.

## Funcionalidades

- Cadastro de tarefas
- Listagem de tarefas
- Busca de tarefas pelo título
- Edição de tarefas
- Exclusão de tarefas

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como executar

1. Faça o download do projeto
2. Abra o arquivo index.html em qualquer navegador
3. Utilize o sistema para gerenciar tarefas
